# Quickstart: 未识别列手动映射

## 测试步骤

### 前置条件
- 后端服务运行在 `http://localhost:8000`
- 前端服务运行在 `http://localhost:5173`
- 准备一个包含 LLM 未识别列头的 Excel 文件（例如包含"首席教授"、"考试方式"等非常规列头）

### 场景 1: 手动映射未识别列

1. 打开前端页面 `http://localhost:5173`
2. 点击上传区域，选择包含未识别列的 Excel 文件
3. 等待分析完成，查看列映射预览
4. 在"未识别的列"区域，找到下拉选择框
5. 为一个未识别列选择系统字段（如将"首席教授"映射到"instructor"）
6. 点击"确认导入"
7. 验证：导入的课程列表中，该列数据正确出现在对应字段中

### 场景 2: 动态过滤已映射字段

1. 上传 Excel 后查看列映射预览
2. LLM 已识别"课程名称→name"
3. 查看任意未识别列的下拉框
4. 验证：下拉框中不包含"name"选项

### 场景 3: 不选择直接导入

1. 上传 Excel 后查看列映射预览
2. 不对任何未识别列做选择（保持"不映射"默认值）
3. 点击"确认导入"
4. 验证：导入成功，未识别列的数据被静默忽略，不影响其他列

### 场景 4: 多列同时手动映射

1. 上传包含多个未识别列的 Excel
2. 为 2-3 个未识别列分别选择不同的系统字段
3. 点击"确认导入"
4. 验证：所有手动映射的列数据都正确导入
