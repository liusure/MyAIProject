# Quickstart: 导入显示优化及推荐和课程表bug修复

## Scenario 1: 导入成功展示条数

1. 启动后端和前端
2. 访问 `/select`，上传一个包含 50 行有效数据的 Excel 文件
3. 确认映射预览正确，点击"确认导入"
4. **预期**: 页面显示"成功导入 50 条"，不显示课程列表
5. 上传一个包含无效学分的文件
6. **预期**: 页面显示失败信息（行号+错误原因），不显示成功条数

## Scenario 2: 推荐功能返回有效方案

1. 上传一个包含课程数据的 Excel 文件
2. 在对话框输入"推荐一些数学相关课程"
3. **预期**: 返回至少一个推荐方案，包含课程名称、学分、匹配度
4. 确认推荐方案中的课程名称与上传文件中的课程一致

## Scenario 3: 周课程表正常显示

1. 获取推荐方案后，向下滚动到方案底部
2. **预期**: 每个推荐方案下方出现周课程表网格
3. 有 schedule 数据的课程显示在对应时间格子中
4. 无 schedule 数据的课程表区域显示"时间未指定"提示

## Scenario 4: 完整导入流程（全量数据）

1. 上传一个包含 1000+ 行的 Excel 文件
2. 确认映射预览，点击"确认导入"
3. **预期**: 成功导入全部 1000+ 条，不只是 3 条
